## What is this?

A web browser you can control from your Scheme repl!

## Dependencies
* gtk3
* webkit2gtk-4.0
* guile-2.2

## How to build
* Compile with `./build.sh`

## How to use
* Run `./schemekit` after compiling and it will launch the browser.
* Run `(open-page "https://example.org")` to see it in action.
